# 🎮 언리얼 엔진 3번 과제

### 📌 프로젝트 설명
- 퍼즐 매니저(`APuzzleManager`)가 월드에 두 종류의 발판(`AItem`, `AMove`)을 자동으로 스폰하고, 퍼즐매니저를 통해 외형(Mesh, Material) 및 타이머 기능(블링크, 랜덤이동, 방향전환)을 통합 제어하는 시스템입니다.

### 🛠️ 구현 기능
1. **APuzzleManager**: 에디터에서 메시/머터리얼 풀을 설정하고 마스터 스위치로 모든 발판의 기능 일괄 제어
    -Settings에서 생성 개수 설정
    -Visuals에서 스태틱매시 및 머터리얼 설정
    -Master Switch에서 기능제어 Reverse는 회전 액터만 적용되는 방향전환 활성화입니다.
2. **AItem**: 회전 기능 및 매니저 명령에 따른 블링크, 랜덤이동, 방향전환 타이머 구동
3. **AMove**: 왕복 이동 기능 및 매니저 명령에 따른 블링크, 랜덤이동 타이머 구동
